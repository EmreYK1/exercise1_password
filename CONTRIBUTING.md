# Contributing Guidelines

Thank you for considering contributing to `@emreyk1/password-utilities`!

## How to Contribute

1. **Bug Reports**: If you find a bug, please create an issue using the [Issue template](ISSUE_TEMPLATE.md).
2. **Feature Requests**: If you have an idea for a new feature, please submit a [Feature Request](FEATURE_REQUEST_TEMPLATE.md).
3. **Pull Requests**:
   - Fork the repository.
   - Create a new branch for your feature or bug fix.
   - Write tests for your changes.
   - Ensure all tests pass by running `npm test`.
   - Submit a pull request using the [Pull Request template](PULL_REQUEST_TEMPLATE.md).

## Development Setup

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Run tests: `npm test`.

## Code Style

Please maintain a consistent code style and ensure your code is well-documented.
